```
TITANIUM_OS/
├── DASHBOARD/          React 19 + Vite + Tailwind v4
│   ├── NeuroMapView    Mappa SVG drill-down immersiva (riferimento estetico)
│   ├── NeuroOSLayout   Vista mappa con dot grid + pulse rings + spring animation
│   ├── CanvasLayout    Vista canvas draggabile per stato progetti
│   ├── LayersView      SINAPSI — database layer dei progetti
│   └── StorieView      Inventario episodi podcast + dataset LLM
├── AUTOMATIONS/        34 automazioni (13 attive, 14 in sviluppo, 7 Content Engine)
├── BRAIN/              STATE.json + KNOWLEDGE/ + milestone tracker
├── CONTENT_ENGINE/     Pipeline auto-generazione episodi da STATE.json
```
